# Task Management Frontend

A modern, responsive dashboard for managing tasks, built with Next.js 14, TypeScript, and Tailwind CSS.

## Features
- Secure Login & Registration
- Interactive Task Dashboard
- Admin Statistics (Admin only)
- Real-time Task Management (CRUD)
- Responsive Design (Dark/Light mode)

## Getting Started
1. `cd frontend`
2. `npm install`
3. Create a `.env.local` file with `NEXT_PUBLIC_API_URL=http://localhost:5000/api/v1`
4. `npm run dev`

## Tech Stack
- Next.js (App Router)
- React
- Tailwind CSS
- Lucide React (Icons)
- Axios
- TypeScript
